Introduction to Git & GitHub:

What is Git?

Git is a version control system used to track changes in files and manage code history.
Version Control Basics
Tracks changes in code or files
Allows multiple people to work on the same project
Helps restore previous versions if mistakes occur

What is GitHub?

GitHub is an online platform that stores Git repositories and allows developers to collaborate.

Features:

*Store code online
*Collaborate with others
*Manage projects
*Share work publicly
